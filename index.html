About

Excel For Efficiency! is an educational primer with the objective of teaching its audience the basics of utilizing a spreadsheet application. This educational primer contains features and attributes which help facilitate the learning of the audience.

Installation

Excel For Efficiency! is developed in PowerPoint with the use of Visual Basic Macros.

Listed below are the steps for installation:

  1. Download from the links in the website

    2. Select your chosen version (i.e. Full Version or Demo Version) from the download links provided in the website. Simply, click on the button and download.

  2. Extract the files and run "unblock.bat"

    3. By default, Microsoft disables the functions of "macros" or the codes within the PowerPoint presentation. By running "unblock.bat", the files are given the necessary permissions to function properly.

  3. Run "ExcelForEfficiency.pptm"

    4. In case a prompt appears saying "Macros has been disabled", simply click on the "Enable" button. Without macros, the program would have no functionality.

Basics

The spreadsheet interface consists of 5 elements, namely,

  1. Name box

    It displays the cell reference or the name of the active cell.

  2. Formula bar

    It can be used for entering and editing data formulas. It also actively displays the content of the active cell.

  3. Sheet tabs

    It tells what is the name of the worksheet you’re currently working with. You can also change to different worksheets by clicking the worksheet file beside it.

  4. Row numbers

    It runs vertically in a worksheet and is identified by a number in the row header.

  5. Column letters

    It runs horizontally in a worksheet and is identified by a number in the column header.

Formulas

Excel uses standard operators for formulas.

  • Plus sign (+) for addition

  • Minus sign (-) for subtraction

  • Asterisk (*) for multiplication

  • Forward slash (/) for division

  • Caret (^) for exponents

An equals sign (=) is used to start a formula. A complex formula will follow the PEMDAS order.

PEMDAS wherein:

Parenthesis
Exponent
Multiplication
Division
Addition
Subtraction

Excel has a feature that allows you to calculate values from specific cell addresses.

Cell addresses are a combination of a column letter and a row number. When incorporated in a formula, these are known as a cell reference. Utilizing cell references will make formulas more accurate.

Functions

A function is a predefined formula in Excel which performs calculations for a specific value using cell references.

=SUM(A1:A5)

A function consists of the equals sign (=), the function name, and one or more arguments. An argument is a range of cell addresses or any input that you provide to a function.

A few of the most commonly used functions are listed below:

  1. SUM

    adds all values of cells in the argument.

  2. AVERAGE

    gets the average of all the values of cells in the argument.

  3. COUNT

    counts the number of cells with numerical data in the argument.

  4. MAX

    gets the highest cell value included in the argument.

  5. MIN

    gets the lowest cell value included in the argument.

Here are some example scenarios wherein you might make use of some of the functions:

=AVERAGE(B2:F2)

Outputs the average of all values in cells B2 to F2
=COUNT(A2:C5)

Outputs the number of all numerical values in cells A2 to C5
=MAX(C1:C10)

Outputs the maximum/highest value found in cells C1 to C10
Sorting

One of the best features of Excel is that you can manage information efficiently by sorting your data alphabetically, numerically, and in many other ways.

  1. Sort Sheet

    Sorts all data in your worksheet by one column. Related information is kept together when sorted.

  2. Sort Range

    Sorts the data in a range of cells, especially helpful when working with a sheet that contains several tables. Only the contents of the specified table will be sorted.

Charts

A chart is a graphical representation of your workbook data which makes it easier to visualize comparisons and trends.

  1. Column

    Uses vertical bars to represent data. Most frequently used for comparing information.

  2. Line

    Ideal for showing trends. The data points are connected by lines, which makes it easier to see the increase or decrease of value over time.

  3. Pie

    Used to compare proportions. Depicts values that make up a percentage of the whole and are shown as slices of a pie.

  4. Area

    Similar to a line chart, except the areas under the lines are shaded.

  5. Surface

    Allows you to display data in 3D landscape. Lets you see a variety of information at the same time. Works best with large data sets.
